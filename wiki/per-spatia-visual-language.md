# PerSpatia 视觉与空间感 (Visual Language)

本文说明 **r3f-reactflow-nextui** 里「2D 画布 + 3D 景深」这一美学**想达成什么**，与具体相机公式无关；实现细节见 [camera-control.md](./r3f-docs/camera-control.md)。

---

## 1. 我们为什么要做 3D 化

目标**不是**让用户像建筑漫游一样 **360° 绕场景飞一圈**。核心目的是 **空间感与纵深**：让用户在编辑信息流、节点图时，仍能联想到「屏幕背后」还有层次、机制与数据在运转——把抽象的系统 **在想象里** 变得可触摸一点。

参考气质上接近 **DATA WING 游戏**：在这个游戏里，叙事里玩家（一只 DATA WING）和 Mather（系统里的一个内置 AI，控制系统的大部分，并指挥调遣所有 DATA WING）是存在于"用户"的一台手机中的，游玩时玩家作为一只在地图里飞行的 DATA WING，长期看到的是 **「平面内容 + 背后深邃的 3D 空间」**——那种深度会自然引导玩家去脑补：深处有什么数据、代码、虚拟机制在运转。PerSpatia 希望借类似的 **plane-on-3D** 语言，唤起我们对「信息世界」的某种原初想象——它只在上个时代自然地存在过，而在今天已经被潮流忽视。它是我们人类初次步入信息时代时，最 intuitive、最自然、最朴素的想象、也是我们在目前跑偏的潮流声量下被长期忽视的、然而却是我们心底里原本最想要的那种，视觉感受。PerSpatia 和 RcA 想要做的是让我们找回这种感觉，找回这种目前只被称为 FUI、然后被人们高高挂起、被贴上不切实际标签的、那种感觉、那种风格。PerSpatia 并不是单纯多一个旋转玩具，或者炫技。

## 2. 与当前实现的差距（诚实状态）

项目组在 **assets**（或等价资源目录）中保留有 DATA WING 的录屏，曾作为把上一代纯 2D 子工程（如 `solution-react-flow`）推进到「类 DATA WING 的 3D 世界」时的**视觉参考**（启动 r3f 子工程迭代时曾把录屏交给模型并直接下指令）。若录屏未进 Git，以本地/团队约定路径为准。

以**当前** r3f 子工程实机画面而言，与 DATA WING 或理想定调相比，仍偏 **毛坯、简陋**：材质、动效、色彩节奏、UI 与景深的层次都还有大量打磨空间。本文档把 **意图** 写清楚，方便以后迭代视觉时对照，而不是宣称已经还原参考作。

## 3. 空间约定与相机预期（与 camera-architecture 对齐）

**世界系里「板子」不是地板，是一面墙：**

- **XY 平面**：**竖直**的，相当于 **一面墙**；React Flow 画布贴在这面墙上（实现里仍是 **Z = 0** 平面）。
- **Y 轴**：**竖直**（沿墙上下）。
- **X 轴**：沿墙水平方向（例如向右）。
- **Z 轴**：**水平纵深**——垂直于这面墙、向场景里「伸进去 / 拉出来」的方向；相机大致在 **Z 正方向一侧** 看向墙上的注视点，和「屏幕外往里看」的纵深一致。

---

## 4. 空间氛围：从虚空到氛围场 (Ambient Field)

我们明确拒绝死寂、冰冷的“纯粹虚空”。PerSpatia 的空间应具备**温度与色彩感**：

- **暖色系基调 (Warm Ambient)**：首选方案是建立一个偏暖色系的、具有空气感的背景（Ambient）。这不只是颜色的填充，而是一种“氛围场”。这种色调应能让人感受到一点点温度，在理科科幻的冷峻中保留一份人性的愉悦感，适合长感官工作。
- **环境景观 (Environmental Landscape - 远期目标)**：作为一种高级演进方向，我们可以尝试描绘出位于星云之中、或是在星球之间的环境景观。这代表了“理科科幻”在视觉表现力上的终极形态——将数学真理置于宏大的宇宙奇观之中。虽然目前受限于实现成本，但这始终是我们的视觉指引。

---

## 5. 美学基调：理科科幻 / Future's Theorist Sci-Fi

在 UI 材质与元素的表达上，PerSpatia 严格遵循 **“理科科幻”** 取向，但在色彩上融入人文温度：

- **拒绝工科拟物**：UI 绝不是现实物理面板的投影。拒绝生硬的厚度、铆钉、机械感滑块。
- **拥抱绝对几何与能量流**：视觉构成应是极简的，质感必须是“非实体”的。例如发光的霓虹矢量线（Neon Vectors）、纯粹的光晕（Bloom）、以及无重量感的玻璃/力场。
- **核心心智隐喻 (Theorist's Desk)**：想象一位未来理论家的桌面。虽然他在推演宇宙中最宏大的真理，但他身处的环境是柔和、有温度且舒适的。**发光的几何实体悬浮在暖色的氛围场中**，这是 PerSpatia 的核心视觉张力。
- **特此排雷**：明确拒绝镜面/反射材质，减少视觉噪点。
- **Least Cues 延伸**：视觉上的“指示词（Cues）”降至最低。不需要多余的边框去强调“这是一个容器”，让发光的边界或能量流转自然地界定空间。
- **实践与演进 (Living Guideline)**：上述审美取向并非死板的教条，而是当前探索的基点（Baseline）。务必将此视为一份会不断进化的活文档。
