# PerSpatia Agent Wiki (Experience Summation)

这是一个面向 AI Agent 的知识库，记录了在不同环境下进行项目开发、调试和验证的最佳实践。

## 目录结构

- [**Cursor Agent 经验**](./cursor-agent/screencast-debugging/README.md) - 针对 Cursor 集成环境的总结。
- [**Gemini CLI Agent 经验**](./gemini-cli-agent/01-browser-debugging.md) - 针对 Gemini CLI 终端环境的总结。

## 维护准则

1. **环境区分**：由于 Agent 工具链差异，新的经验必须明确标记适用环境。
2. **亲自验证**：所有记录的方案必须由 Agent 在当前 Session 中亲自验证可行后再记录。
3. **数值优先**：在 3D 场景下，优先记录可读的快照数值而非纯视觉描述。
